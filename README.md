This project focuses on a fictional organization that has detected unusual activity in its systems. The tasks included:

- Analyzing firewall logs, system logs, and authentication logs

- Identifying unauthorized access attempts

- Reviewing suspicious IP connections

- Mapping IOCs to potential threat actors or techniques

- Determining the timeline of the incident using event correlation

- Evaluating the security impact on the organization

This provides hands-on experience similar to real-world SOC investigations.

# üìù Methods & Techniques

This project uses a structured cybersecurity investigation approach:

1. Detection & Initial Review

Examined security alerts

Identified suspicious events and connections

2. Log Analysis

Reviewed authentication logs for failed and successful attempts

Analyzed firewall logs for unauthorized traffic

Inspected system logs for anomalies and process activity

3. Incident Reconstruction

Built a timeline of attacker activity

Mapped login attempts, privilege escalations, and data access patterns

4. IOC Identification

Extracted suspicious IPs, file names, user accounts, and hash values

Compared findings against known threat intelligence sources

5. Impact Assessment

Evaluated compromised accounts or systems

Determined whether data was accessed or exfiltrated

6. Recommendations

Suggested short-term and long-term remediation steps

Provided security hardening measures based on NIST CSF & CIS Controls
